Concepts of Machine Learning

Components of Machine Learning

  • Hypothesis space & Learning Algorithm

    在决策空间中利用学习算法找到最优的决策，讨论的决策空间往往是一系列备选函数的集合，学习算法试图找到其中最符合我们要求的函数(a prediction \(h\))

    我们期望决策空间只包含具有正则性的函数，如对于分类问题，决策空间包含线性超平面

  • Loss Function 衡量我们找到的\(h\)与实际结果的差异

Machine Learning Scenarios

  • Classification(分类)
  • Clustering(聚类)
  • Regression(回归)

Evaluation Metrics

T: True, F: False, P: Positive, N: Negative - Classification - Precision (\(\frac{TP}{TP+FP}\)) - Recall (\(\frac{TP}{TP+FN}\)) - Accuracy (\(\frac{TP+TN}{TP+TN+FP+FN}\)) - Error (I - Accuracy) - Confusion Matrix

  • Regression
    • Error(Squared, Absolute, Squared Logarithmic, Absolute Percentage)

算法

列举了一些常用的算法（主要是分类的）

K-Nearest Neighbors(KNN)

为预测x的类型，寻找k个与x最近的neighbors，以其中的大多数的类型作为x的类型

随着k的增大，噪音被抹平，决策边界变得平滑，越不容易overfit

k作为超参数，可以利用validation set决定，通常取奇数

优点 - 可以学习复杂函数 - 不丢失原始数据信息 - 可以学习非常大的数据集及类数目

缺点 - 推断时低效 - 依赖于特征选择

Linear Classification

Logistic Regression/Softmax Regression

名为回归，实际上是用回归的方法获得一个连续的分类的概率，再将其映射为一个离散的类别

Softmax Regression是将Logistic Regression拓展到多分类

Support Vector Machine

支持向量机，一种二分类模型

基本思想是找到能够正确划分数据并且几何间隔最大的超平面，与LR的区别在于它要求集合间隔最大，因此只有一个符合要求的超平面，而LR能找到无数个符合要求的超平面，和LR相比，本质在于Loss Function不同

线性SVM只依赖于数据的边界，而不依赖数据的全局分布，因此能解决数据unbalance的情况

可以引入核函数解决非线性问题

Decision Tree

根据Feature，多层决策来得到最终的分类，会得到一系列平行于决策空间坐标轴的决策面

Random Forest
