Hexo添加LaTeX支持

Reference: 给Hexo主题添加LaTeX公式支持，写得非常清晰，本文仅做个人记录

请先查看使用的主题是否已经支持\(\LaTeX\)相关配置，如有，参考主题的Readme自行配置即可

如果自带主题并未支持\(\LaTeX\)，则手动安装相关依赖。比如我使用的Minima太过精简，缺少相关配置

步骤

  1. 在themes/<theme-name>/layout中合适的位置新建文件mathjax.ejs，内容如下 

    <% if (theme.mathjax.enable){ %>
    <script type="text/x-mathjax-config">
      MathJax.Hub.Config({
          tex2jax: {
            inlineMath: [ ['$','$'], ["\\(","\\)"] ],
            processEscapes: true
          }
        });
      </script>

    <script type="text/x-mathjax-config">
      MathJax.Hub.Config({
            tex2jax: {
              skipTags: ['script', 'noscript', 'style', 'textarea', 'pre', 'code']
            }
          });
      </script>

    <script type="text/x-mathjax-config">
      MathJax.Hub.Queue(function() {
              var all = MathJax.Hub.getAllJax(), i;
              for(i=0; i < all.length; i += 1) {
                  all[i].SourceElement().parentNode.className += ' has-jax';
              }
          });
      </script>

    <script type="text/javascript" src="<%- theme.mathjax.cdn %>"></script>
<% } %>
    该模板文件用于通过cdn引入mathjax，并做语法解析等相关操作

  2. 在主题的配置文件中添加 

    mathjax:
  enable: true
  cdn: https://cdnjs.cloudflare.com/ajax/libs/mathjax/2.7.6/MathJax.js?config=TeX-AMS-MML_HTMLorMML
    用于控制启用mathjax，并指定cdn地址

  3. 在themes/<theme-name>/layout/post.ejs中添加 

    <% if (page.mathjax == true){ %>
    <%- partial('/your/path/to/mathjax') %>
<% } %>
    用于引入mathjax相关模板

  4. 在需要\(\LaTeX\)的文章头部添加mathjax: true以启用mathjax，如果觉得这样太麻烦，可以将第三步中的if判断去掉

  5. 替换渲染引擎, 

    npm uninstall hexo-math --save
npm uninstall hexo-renderer-marked --save
npm install hexo-renderer-pandoc --save
    同时需要在电脑上安装pandoc2.0及以上版本
